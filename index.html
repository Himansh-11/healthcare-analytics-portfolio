‚ùó Problem Statement

Hospital readmissions within 30 days cost Medicare billions annually and indicate gaps in care quality. Despite risk-adjustment models, significant variation exists across hospitals and states. Healthcare administrators lack comprehensive tools to:

  • Identify geographic patterns in readmission performance
  • Compare hospital-level outcomes against national benchmarks
  • Quantify financial impact of excess readmissions
  • Prioritize improvement initiatives based on ROI

üí° Key Findings

  • $207.42M Annual Opportunity: Identified potential Medicare savings if all hospitals achieved top-quartile performance (excess ratio < 0.87)
  • 116% Performance Variation: Hospital excess readmission ratios range from 0.48 (52% better than benchmark) to 1.64 (64% worse than benchmark)
  • Geographic Disparities: State-level performance varies from 0.92 to 1.03, suggesting regional differences in care quality and infrastructure
  • Weak Satisfaction Correlation: Patient satisfaction shows minimal relationship with readmission rates, indicating distinct quality dimensions
  • Heart Failure = Highest Cost: Identified highest-cost condition for targeted intervention programs
  • 56% Above Benchmark: Majority of hospitals exceed national readmission expectations, highlighting systemic improvement opportunities

üéØ Strategic Recommendations

  • Targeted Interventions: Focus improvement efforts on heart failure readmissions (highest cost driver) with evidence-based discharge protocols
  • Best Practice Sharing: Implement knowledge transfer programs from top-performing states (ratio < 0.90) to underperforming regions
  • High-Impact Hospitals: Prioritize the top 100 hospitals by excess cost for intensive quality improvement initiatives - potential for maximum ROI
  • Condition-Specific Protocols: Develop standardized post-acute follow-up procedures for high-readmission conditions
  • State-Level Collaboration: Share resources and strategies among hospitals within underperforming states to address systemic issues

üõ†Ô∏è Technical Implementation

Data Sources

  • CMS Hospital Compare (2024): Hospital Readmissions Reduction Program data
  • Tables: Hospital Readmissions (11,927 records), Hospital General Info (2,497 hospitals), Patient Satisfaction (3,975 records)

Data Model

  • Schema: Star schema with Hospital General Info as dimension table
  • Relationships: One-to-many from General Info to Readmissions/Satisfaction via Hospital ID
  • Cardinality: Properly configured (1:*) with bidirectional cross-filtering where needed

Key DAX Measures

  • Dynamic Filtering: Selected Hospitals, Best/Worst in Selection using ALLSELECTED
  • Rankings: State rankings with RANKX, performance quartiles with SWITCH
  • Financial Calculations: Cost by condition, excess cost, ROI projections
  • Aggregations: AVERAGEX for state-level metrics, CALCULATE with filters for segmentation

Visualization Techniques

  • Geographic Analysis: ArcGIS map with color saturation by performance
  • Correlation Analysis: Scatter plot with dynamic legends and tooltips
  • Financial Storytelling: Waterfall chart showing cost buildup, donut chart for segmentation
  • Interactivity: Cross-page navigation, synchronized slicers, drill-through capabilities
